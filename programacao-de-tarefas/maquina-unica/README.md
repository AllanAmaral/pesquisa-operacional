# Máquina Única (1)

Esta pasta reúne implementações para o problema de programação de tarefas no ambiente de máquina única.

As notações e a nomenclatura utilizada nas pastas `modelos-exatos`, `heuristicas` e `instancias` foram ajustadas para manter alinhamento com a literatura internacional. Para aprofundamento nos fundamentos teóricos do problema, recomenda se a consulta às obras que tratam especificamente do problema de programação de tarefas.
